      Deep Dive
      Einstellungen

      Alle Analysen laufen kostenlos über Claude Code auf deinem Mac (Claude-Abo). Das Dashboard zeigt nur an und startet den Mac-Lauf per Knopf.

      Technik-Feed (TradingView-Bridge)

      Raw-URL deines GitHub-Gists mit der tech.json, die Claude Code am Mac über die TradingView-Bridge erzeugt. Sobald hinterlegt, fließt die Technik-Analyse mit in den Bias ein und das Dashboard warnt oben, wenn die Verbindung abbricht oder die Daten veraltet sind.

      GitHub-Token (Fernstart der Mac-Analyse)

      Fine-grained Token mit nur „Gists"-Berechtigung (github.com → Settings → Developer settings). Damit startet der „Analyse aktualisieren"-Button die Technik-Analyse + den Tokyo-Scan auf deinem Mac (Wächter-Skript muss dort laufen). Wird nur lokal gespeichert und nur an api.github.com gesendet.
